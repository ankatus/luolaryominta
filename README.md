# Käyttöohjeet
q,w,e,a,s,d,z,x,c liikuttaa pelaajaa  
ruudulla olevista napeista voi valita pelimoodin  
vihreät palikat antaa yhden hp:n, keltanen palikka voittaa pelin  
punaiset on hirviöitä, tekevät yhden hp:n vahinkoa
